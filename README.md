# 📊 Analyze Social Media Engagement to Uncover Brand Performance Insights

## 📌 Project Overview
This project analyzes social media engagement data to uncover brand performance insights using SQL, Python, and Power BI.

## 📁 Files Included
- **CapsoneDashboardUpdated.pbix** – Power BI Dashboard
- **SQLBasicQueries.sql** – Basic SQL queries
- **SQLObjectives1-4.sql** – SQL analysis objectives
- **Social_Media_Engagement.ipynb** – Jupyter Notebook analysis
- **audience_demographics.xlsx** – Audience data
- **engagement.xlsx** – Engagement metrics
- **post.xlsx** – Post performance data
- **creator.xlsx** – Creator data
- **sponsorship.xlsx** – Sponsorship data

## 🛠️ Tools Used
- Python (Jupyter Notebook)
- SQL
- Microsoft Power BI
- Microsoft Excel
